# GenLayer Testnet Bradbury — Builder Guide

A practical guide to onboarding on GenLayer and earning Builder Points.

## What is GenLayer?
GenLayer is the first intelligent blockchain where AI directly participates in consensus. Smart contracts can access the internet and use LLMs to make decisions on-chain.

## Step 1 — Setup
- Install Rabby or MetaMask
- Create an account on [GenLayer Portal](https://portal.genlayer.foundation/)
- Connect your wallet and complete your profile (GitHub, Twitter, Discord)

## Step 2 — Get Studio Access
- Go to [GenLayer Studio](https://studio.genlayer.com/)
- Connect your wallet and request whitelist access
- Wait for approval (usually a few hours)

## Step 3 — Deploy Your First Contract
- Open `storage.py` in the Studio
- Click **Deploy** and confirm in your wallet
- Save your transaction hash as proof of deployment

## Step 4 — Submit Contributions
- Go to Portal → Submit Contribution
- Select Builder → Projects & Milestones
- Add your tx hash and GitHub repo as evidence
